# Modern Professional Portfolio & Case Study

## 📌 Project Overview 
This project is a high-end, responsive personal portfolio platform designed for creative professionals and engineers. It serves as a comprehensive digital hub, integrating a project showcase, a technical blog, and a digital storefront into a high-performance, seamless user experience.

---

**Project Context:**
- **Client:** This project was developed for a client in **Egypt**.
- **Year:** Completed in **2021**.
- **Code Privacy:** The source code and specific logic are archived and kept private as per the **client's explicit request**.

---

## ✨ Key Features
* **Dynamic Theme Switching**: A built-in system allowing users to toggle between **Light and Dark modes** instantly, with persistent settings.
* **Interactive UX Elements**: 
    * **Magnetic Cursor**: A custom-coded interactive cursor that reacts to hoverable elements.
    * **Smooth Page Transitions**: Powered by **PJAX** (PushState + AJAX) for fast, app-like navigation without full page reloads.
* **Visual Excellence**:
    * **WebGL Backgrounds**: Uses a `js-webgl` canvas for high-performance, interactive background animations.
    * **Parallax Effects**: Advanced scroll-based depth animations for project previews.
* **Content Modules**:
    * **Portfolio Gallery**: High-fidelity grid system for project case studies.
    * **Article Engine**: A dedicated blog layout optimized for professional industry insights.
    * **Digital Store**: A front-end commerce interface for products and assets.
* **Fully Responsive**: Optimized using custom grid systems to ensure a perfect experience across mobile, tablet, and desktop devices.
* **Bi-directional Support**: Architected to support both **English (LTR)** and **Arabic (RTL)** layouts seamlessly.

## 🛠️ Technical Stack
* **Front-end**: HTML5, CSS3, JavaScript (ES6+).
* **Libraries**: 
    * **jQuery**: Used for DOM manipulation and core UI logic.
    * **Swiper/Vendor Scripts**: For high-performance touch sliders and carousels.
    * **FontAwesome & Custom Icon Fonts**: Specialized iconography for creative brands.
* **Performance**:
    * **Lazy Loading**: Backgrounds and images load on demand to optimize speed.
    * **GPU Acceleration**: Scripts specifically optimized to run on high-performance GPUs for smooth animations.
---

## 📂 Project Structure
* `index.html`: Main landing page and brand introduction.
* `about.html`: Professional background and expertise summary.
* `projects.html`: Interactive portfolio gallery.
* `articles.html`: Blog feed and knowledge sharing section.
* `store.html`: Digital storefront interface.
* `contact.html`: Lead generation and social integration.
* `js/main.js` & `js/components.js`: Core logic for UI interactions, cursor effects, and theme management.
* `css/main.css`: Comprehensive styling including grid systems and theme variables.
